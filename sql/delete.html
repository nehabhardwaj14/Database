DELETE COMMAND

  • The DELETE statement is used to delete rows from a table

  • To remove a specific row from a table you should use WHERE condition.

        
SYNTAX:

        DELETE FROM table_name WHERE condition;

EXAMPLE:

         mysql> delete from student where section="b";
         Query OK, 5 rows affected (0.01 sec)

TRUNCATE COMMAND

  • The TRUNCATE statement: it is used to delete all the rows from the table and free the containing space.

  • TRUNCATE a table, the table structure remains the same, so you will not face any of the above problems.

            
SYNTAX:

            TRUNCATE TABLE TABLE_NAME;

EXAMPLE:

TABLE SAMPLE:-

            mysql> select*from sample;
            +------+--------+
            | id   | name   |
            +------+--------+
            |  101 | harman |
            |  102 | raju   |
            |  103 | geet   |
            |  104 | kiara  |
            |  105 | anshu  |
            +------+--------+
            5 rows in set (0.00 sec)

            mysql> truncate table sample;
            Query OK, 0 rows affected (0.06 sec)
        
              
            
TABLE AFTER TRUNCATE:-

            
            mysql> select*from sample;
            Empty set (0.00 sec)
            
            mysql> desc sample;
            +-------+-------------+------+-----+---------+-------+
            | Field | Type        | Null | Key | Default | Extra |
            +-------+-------------+------+-----+---------+-------+
            | id    | int         | YES  | UNI | NULL    |       |
            | name  | varchar(20) | YES  | UNI | NULL    |       |
            +-------+-------------+------+-----+---------+-------+
            2 rows in set (0.00 sec)
