INTRODUCTION

SQL is a short-form of the structured query language, and it is pronounced as S-Q-L or sometimes as See-Quell. This database language is mainly designed for maintaining the data in relational database management systems. It is a special tool used by data professionals for handling structured data (data which is stored in the form of tables). It is also designed for stream processing in RDSMS

You can easily create and manipulate the database, access and modify the table rows and columns, etc. This query language became the standard of ANSI in the year of 1986 and ISO in the year of 1987. If you want to get a job in the field of data science, then it is the most important query language to learn. Big enterprises like Facebook, Instagram, and LinkedIn, use SQL for storing the data in the back-end.

SQL Data Types

Data Type Description
INT Integer: Whole numbers
FLOAT Floating-point numbers with decimal precision
DOUBLE Double-precision floating-point numbers
DECIMAL(p, s) Fixed-point numbers with precision p and scale s
CHAR(n) Fixed-length character string with a maximum length of n
VARCHAR(n) Variable-length character string with a maximum length of n
TEXT Variable-length character string with no specified maximum length
DATE Date (YYYY-MM-DD format)
TIME Time (HH:MM:SS format)
DATETIME Date and time combination
BOOLEAN Represents true or false values
BINARY(n) Fixed-length binary string with a maximum length of n
VARBINARY(n) Variable-length binary string with a maximum length of n
BLOB Binary Large Object, for storing large binary data
JSON Stores JSON (JavaScript Object Notation) data
ARRAY Stores an array of elements
