GRANT

In MySQL, the GRANT statement is used to grant specific privileges to database users or roles.

These privileges can include various permissions such as SELECT, INSERT, UPDATE, DELETE, and more. Here's the basic syntax:

        
SYNTAX:

        GRANT privileges ON database_name.table_name TO 'user'@'host';

EXAMPLE 1:

        Example to grant SELECT privileges on all tables in the 
        school database to a user named 'john' from localhost:

        mysql>GRANT SELECT ON school.* TO 'john'@'localhost';
      
        
EXAMPLE 2:

        Example to grant multiple privileges at once:

        mysql>GRANT SELECT, INSERT, UPDATE ON employees.* TO 'john'@'localhost';

EXAMPLE 3:

        Example to grant all privileges on all tables in the 
        school database to a user named 'john' from localhost:

        mysql>GRANT ALL PRIVILEGES ON school.* TO 'john'@'localhost';
      
        
EXAMPLE 4:

         Example to grant all privileges on all databases to a user named 'john' from localhost:

       mysql> GRANT ALL PRIVILEGES ON *.* TO 'john'@'localhost';

REVOKE

In MySQL, the REVOKE statement is used to revoke previously granted privileges from users or roles.

It effectively removes specific privileges from a user, limiting their access to certain databases, tables, or columns.

        
SYNTAX:

        REVOKE privileges ON database_name.table_name FROM 'user'@'host';
      
        
EXAMPLE 1:

        Example to revoke SELECT privileges on all tables in the school database 
        to a user named 'john' from localhost:

        mysql>REVOKE SELECT ON school.* TO 'john'@'localhost';
      
        
EXAMPLE 2:

        Example to revoke multiple privileges at once:

        mysql>REVOKE SELECT, INSERT, UPDATE ON school.* TO 'john'@'localhost';
      
        
EXAMPLE 3:

        Example to revoke all privileges on all tables in the school database
        to a user named 'john' from localhost:

        mysql>REVOKE ALL PRIVILEGES ON school.* TO 'john'@'localhost';
      
        
EXAMPLE 4:

        Example to revoke all privileges on all databases to a user named 'john' from localhost:

        mysql>REVOKE ALL PRIVILEGES ON *.* TO 'john'@'localhost';
