PATTERN MATCHING

  • LIKE clause is used to perform the pattern matching task in SQL.

  • A WHERE clause is generally preceded by a LIKE clause in an SQL query.

  • LIKE clause searches for a match between the patterns in a query with the pattern in the values present in an SQL table. If the match is successful, then that particular value will be retrieved from the SQL table.

  • LIKE clause can work with strings and numbers.

The LIKE clause uses the following symbols known as wildcard operators in SQL to perform this pattern-matching task in SQL.

  • To represent zero, one or more than one character, % (percentage) is used.

  • To represent a single character _ (underscore) is used.

        
SYNTAX:

        SELECT ColumnName1, ColumnName2 FROM TableName WHERE ColumnName LIKE [Expression];

EXAMPLE 1:

        mysql> select roll_no,marks,name from student where name like "%a";
        +---------+-------+-------+
        | roll_no | marks | name  |
        +---------+-------+-------+
        |       1 |    98 | Neha  |
        |       3 |    80 | Asha  |
        |       4 |    80 | Riya  |
        |       5 |    40 | Priya |
        +---------+-------+-------+
        4 rows in set (0.00 sec)
     
        
EXAMPLE 2:

        mysql> select name,age,marks from student where name like "%a_";
        +--------+------+-------+
        | name   | age  | marks |
        +--------+------+-------+
        | Misbah |   19 |    89 |
        | Aman   |   19 |    89 |
        | Pawan  |   19 |    75 |
        +--------+------+-------+
        3 rows in set (0.00 sec)
     
        
EXAMPLE 3:

        mysql> select name,section,fees,marks from student where name like "%e__";
        +------+---------+-------+-------+
        | name | section | fees  | marks |
        +------+---------+-------+-------+
        | Neha | A       | 24000 |    98 |
        +------+---------+-------+-------+
        1 row in set (0.00 sec)
         
    
        
EXAMPLE 4:

        mysql> select name,section,fees,marks from student where name like "____";
        +------+---------+-------+-------+
        | name | section | fees  | marks |
        +------+---------+-------+-------+
        | Neha | A       | 24000 |    98 |
        | Asha | A       | 24000 |    80 |
        | Riya | B       | 24000 |    80 |
        | Anil | A       | 24000 |    20 |
        | Aman | B       | 24000 |    89 |
        +------+---------+-------+-------+
        5 rows in set (0.00 sec)
     
        
EXAMPLE 5:

        mysql> select name,section,fees,marks from student where name like "__s%";
        +--------+---------+-------+-------+
        | name   | section | fees  | marks |
        +--------+---------+-------+-------+
        | Misbah | B       | 24000 |    89 |
        +--------+---------+-------+-------+
        1 row in set (0.00 sec)
