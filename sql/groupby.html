GROUP BY CLAUSE

  • The Group By clause is used to arrange similar kinds of records into the groups in the Structured Query Language.

  • The Group by clause in the Structured Query Language is used with Select Statement.

  • Group by clause is placed after the where clause in the SQL statement.

  • The Group By clause is specially used with the aggregate function,like, max (), min (), avg (), sum (), count ().

        
SYNTAX:

        SELECT * FROM TABLENAME GROUP BY COLUMN_NAME;
    
            
EXAMPLE 1:

        mysql> select age from student group by age;
        +------+
        | age  |
        +------+
        |   18 |
        |   19 |
        |   20 |
        +------+
        3 rows in set (0.00 sec)
    
        
EXAMPLE 2:

        mysql> select section from student group by section;
        +---------+
        | section |
        +---------+
        | A       |
        | B       |
        +---------+
        2 rows in set (0.00 sec)
    
        
EXAMPLE 3:

        mysql> select marks,count(marks) from student group by marks;
        +-------+--------------+
        | marks | count(marks) |
        +-------+--------------+
        |    98 |            1 |
        |    90 |            2 |
        |    80 |            2 |
        |    40 |            1 |
        |    89 |            2 |
        |    20 |            1 |
        |    75 |            1 |
        +-------+--------------+
        7 rows in set (0.00 sec)

HAVING CLAUSE

  • we will use the HAVING clause with the Group By clause for column conditions.

        
SYNTAX:

        SELECT * FROM TABLENAME GROUP BY COLUMNNAME HAVING CONDITION; 
    
        
EXAMPLE 1:

        mysql> select marks from student group by marks  having min(marks)<70;
        +-------+
        | marks |
        +-------+
        |    40 |
        |    20 |
        +-------+
        2 rows in set (0.00 sec)
