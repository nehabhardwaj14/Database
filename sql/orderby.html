ORDER BY CLAUSE

  • Order by clause is used to arrrange data in sorted order.

  • By default order by clause shows the data in ascending order as per the mention column_name.

  • To see data in descending order use DESC

        
SYNTAX:

         SELECT * FROM TABLE_NAME ORDER BY COLUMN_NAME ASC/DESC;
    
         
EXAMPLE:

         mysql> select name,age from student order by marks desc;
         +----------+------+
         | name     | age  |
         +----------+------+
         | Neha     |   18 |
         | Abhishek |   19 |
         | Misbah   |   19 |
         | Aman     |   19 |
         | Asha     |   19 |
         | Riya     |   20 |
         | Pawan    |   19 |
         | Priya    |   18 |
         | Anil     |   20 |
         +----------+------+
         10 rows in set (0.00 sec)
