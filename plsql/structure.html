BLOCK STRUCTURE

In PL/SQL, code is organized into blocks. Each block has a specific structure consisting of declarations, executable statements, exception handling, and other optional parts. Here's the general structure of a PL/SQL block:

A block is a fundamental structure used to organize code. A block consists of several sections:

1.Declaration Section: This section is optional and is used to declare variables, constants, cursors, and other program objects that will be used within the block. Declarations are made using the DECLARE keyword.

                
EXAMPLE:

                DECLARE
                x INTEGER;
                y VARCHAR2(50);
                -- Additional declarations go here

2.Executable Section (or Main Section):This section is mandatory and contains the executable statements of the PL/SQL block.

It can include SQL queries, procedural statements (such as loops, conditional statements, etc.), and calls to other procedures or functions. The executable section begins with the BEGIN keyword and ends with the END keyword.

                    
EXAMPLE:

                    BEGIN
                    x := 10; -- Assignment statement
                    y := 'Hello, World!'; -- Assignment statement
                    -- Additional executable statements go here
                   END;

3.Exception Handling Section: This section is optional and is used to handle exceptions (errors) that may occur during the execution of the block.

You can specify different exception handlers to deal with specific types of errors. The exception handling section begins with the EXCEPTION keyword.

                            
EXAMPLE:

                            EXCEPTION
                            WHEN others THEN
                                -- Exception handler code goes here

Here's a simple example of a PL/SQL block demonstrating these sections:

EXAMPLE:

                            DECLARE
                            -- Declaration Section
                            x INTEGER := 10;
                       BEGIN
                            -- Executable Section
                            IF x > 0 THEN
                                    DBMS_OUTPUT.PUT_LINE('x is positive.');
                            ELSE
                                    DBMS_OUTPUT.PUT_LINE('x is non-positive.');
                             END IF;
                       EXCEPTION
                            -- Exception Handling Section
                            WHEN others THEN
                                DBMS_OUTPUT.PUT_LINE('An error occurred: ' || SQLERRM);
                       END;

  • The DECLARE section declares an integer variable x and initializes it with the value 10.

  • The BEGIN section contains an IF statement that checks if x is greater than 0 and prints a message accordingly.

  • The EXCEPTION section handles any exceptions that occur during the execution of the block, printing an error message if any exception is caught.
