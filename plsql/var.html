VARIABLES AND CONSTANTS

In PL/SQL (Procedural Language/Structured Query Language), variables and constants play crucial roles in storing and manipulating data within programs. Here's a breakdown of each:

VARIABLES

  • 1.DECLARATION: Variables are declared using the DECLARE keyword followed by the variable name, data type, and optional default value.

    • EXAMPLE:

                    DECLARE
                my_variable NUMBER := 10;

  • 2.SCOPE: Variables can be declared at different scopes - local, global, or block-level.

  • 3.DATA TYPE: PL/SQL supports various data types for variables including NUMBER, VARCHAR2, DATE, BOOLEAN, etc.

  • 4.ASSIGNMENT: Values can be assigned to variables using the := assignment operator.

    • EXAMPLE:

                    my_variable := 20;

  • 5.USAGE: Variables are used to store and manipulate data within PL/SQL blocks or procedures.

  • 6.MUTUABLE: Variables are mutable, meaning their values can be changed during program execution.

CONSTANT

  • 1.DECALARATION: are declared using the CONSTANT keyword followed by the constant name, data type, and value.

    • EXAMPLE:

                    CONSTANT pi NUMBER := 3.14;

  • 2.SCOPE: Constants are usually declared at the beginning of a program or a package and have global scope within that program or package.

  • 3.DATA TYPE: Constants can have data types similar to variables.

  • 4.VALUE ASSIGNMENT: Constants must be assigned a value at the time of declaration and cannot be changed afterward.

  • 5.USAGE: Constants are used for values that remain constant throughout the execution of a program, such as mathematical constants or configuration settings

EXAMPLE:

            DECLARE
            -- Variable declaration
            my_variable NUMBER := 10;
            
            -- Constant declaration
            CONSTANT pi NUMBER := 3.14;
        BEGIN
            -- Using variables and constants
            my_variable := my_variable + 5;
            DBMS_OUTPUT.PUT_LINE('Updated value: ' || my_variable);
            DBMS_OUTPUT.PUT_LINE('Value of pi: ' || pi);
        END;
